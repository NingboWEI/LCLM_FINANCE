# LCLM_FINANCE

A benchmark for Long-context Language Model in Finance domain

## What it is

A fully automated, large-scale benchmark for evaluating long-context LLMs on cross-document financial numerical reasoning tasks, built from `EDGAR XBRL data` with `company financial report` and evaluated via executable `SQL ground truth`.

It has:

- 1️⃣ Fully automated scoring with scalable avalibity

- 2️⃣ Cross-material reasoning

- 3️⃣ Self-evolution: can keep update with latest finance data automatically
